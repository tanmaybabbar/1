# First Java Program: Variable & Data Type Practice

A simple Java application showcasing the foundational concepts of the language. This project was built to practice handling various data types and implementing basic conditional logic.

## 🚀 What I Practiced
* **Data Types**: Implemented primitive types (`int`, `double`, `char`, `boolean`) and reference types (`String`).
* **Output & Concatenation**: Combined static text with variables using `System.out.println()`.
* **Control Flow**: Used an `if-else` conditional block to dynamically change terminal output based on a boolean value.

## 💻 Tech Stack
* **Language**: Java
* **IDE**: IntelliJ IDEA

## 📄 Sample Output
When executed, the program simulates a simple order confirmation system in the console:
`Hello Tanmay your order is placed successfully.`
`You have successfully made payment for bugatti chiron`
`There is a bugatti chiron available in stock for dispatch in 3 days!`
